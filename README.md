# SakuMuslim

**Aplikasi Muslim Modern yang Dirancang untuk Kenyamanan Ibadah Sehari-hari**

---

## Tentang SakuMuslim

SakuMuslim hadir sebagai pendamping ibadah yang ringan, cepat, dan sepenuhnya privat. Dirancang untuk umat Muslim Indonesia yang menginginkan akses mudah terhadap fitur-fitur ibadah utama tanpa gangguan iklan, tanpa pelacakan data, dan tanpa keharusan membuat akun.

Setiap data yang Anda masukkan dan simpan tetap berada di perangkat Anda. Tidak ada server yang menyimpan informasi pribadi Anda. Privasi Anda adalah prioritas utama kami.

---

## Fitur Utama

### Al-Qur'an Digital
- 114 surah lengkap dengan teks Arab dan terjemahan Bahasa Indonesia (Kemenag)
- Pencarian surah berdasarkan nama atau nomor
- Penanda terakhir dibaca yang tersimpan otomatis
- Pengaturan ukuran font Arab dan terjemahan

### Jadwal Sholat
- Jadwal sholat harian untuk 12 kota di Indonesia
- Metode kalkulasi Kementerian Agama RI
- Alarm otomatis 2 menit sebelum waktu sholat dengan suara adzan
- Penunjuk waktu sholat yang sedang berlangsung

### Tasbih Digital
- Pilihan target: 33, 34, 99, atau jumlah custom
- Animasi visual dan getaran haptic saat berhitung
- Pilihan dzikir: Subhanallah, Alhamdulillah, Allahu Akbar, Laa ilaaha illallah, Astaghfirullah

### Kalkulator Zakat
- Perhitungan zakat maal 2.5%
- Enak kategori harta: emas, perak, uang tunai, tabungan, investasi, properti
- Pengecekan nisab otomatis berdasarkan harga emas dan perak

### Hadist
- 8 kitab hadist utama dari API Ahmad Sanusi
- Terjemahan Bahasa Indonesia
- Navigasi per nomor hadist dengan pagination

### Penunjuk Arah Kiblat
- Kompas real-time berbasis sensor perangkat
- Perhitungan bearing Kaaba dari lokasi Anda
- Animasi visual petunjuk arah

### Mode Gelap
- Tema gelap yang nyaman untuk membaca malam hari
- Beralih otomatis mengikuti pengaturan sistem perangkat

## Prinsip Pengembangan

SakuMuslim dibangun di atas tiga prinsip utama:

**Privasi Absolutut** — Tidak ada akun, tidak ada server, tidak ada pelacakan. Semua data tetap di perangkat Anda.

**Tanpa Iklan** — Aplikasi ini bebas dari iklan apa pun. Fokus Anda saat membaca Al-Qur'an atau beribadah tidak akan terganggu.

**Ringan dan Cepat** — Dirancang untuk bekerja dengan baik di berbagai perangkat Android, termasuk perangkat dengan spesifikasi menengah.

## Instalasi

### Android (APK)
Download file APK dari halaman Release, lalu instal di perangkat Anda. Pastikan opsi "Sumber Tidak Dikenal" diaktifkan di pengaturan keamanan perangkat.

### iOS (PWA)
1. Buka aplikasi Safari di iPhone Anda
2. Akses URL deployment PWA
3. Ketuk tombol Share lalu pilih "Add to Home Screen"
4. Aplikasi akan muncul di layar utama seperti aplikasi native

### Web
Aplikasi juga dapat diakses langsung melalui browser di alamat deployment.
