# Font Installation Instructions

## Required Fonts

### Poppins (Latin Font)
Download from: https://fonts.google.com/specimen/Poppins
Required weights: 100, 200, 300, 400, 500, 600, 700, 800, 900

### Amiri (Arabic Font)
Download from: https://fonts.google.com/specimen/Amiri
Required weights: 400, 700

## Installation Steps

1. Download the fonts from the links above
2. Extract the .ttf files
3. Place them in this directory (assets/fonts/)
4. The app will automatically use them

## File Names Required

Poppins:
- Poppins-Thin.ttf (100)
- Poppins-ExtraLight.ttf (200)
- Poppins-Light.ttf (300)
- Poppins-Regular.ttf (400)
- Poppins-Medium.ttf (500)
- Poppins-SemiBold.ttf (600)
- Poppins-Bold.ttf (700)
- Poppins-ExtraBold.ttf (800)
- Poppins-Black.ttf (900)

Amiri:
- Amiri-Regular.ttf (400)
- Amiri-Bold.ttf (700)
